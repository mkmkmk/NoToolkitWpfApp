# NoToolkitWpfApp
============
Created by Mariusz K.

Parts are taken from:

SharpDX Samples 
    http://github.com/sharpdx/SharpDX.git
    
SharpDX DirectX11 Tutorial 
    http://www.notjustcode.it 
    
SharpDX.Toolkit
    https://github.com/sharpdx/Toolkit
